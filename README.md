# Lógica de Programação com Java

Este repositório foi criado para documentar e organizar a minha evolução nos estudos de fundamentos de programação e lógica utilizando a linguagem Java. Aqui estão reunidos exercícios práticos focados na construção de uma base técnica sólida.

## Tecnologias Utilizadas
* **Java** (JDK 17 ou superior)
* **Git & GitHub** (Controle de versão e documentação)

## 📁 Estrutura do Repositório

O projeto está dividido em módulos cronológicos e temáticos para facilitar a navegação:

* **01. Fundamentos:** Exercícios com operadores aritméticos, lógicos, de comparação, atribuição e conceitos de incremento/decremento.
* **02. Estruturas Condicionais:** Aplicação prática de tomada de decisão no código utilizando estruturas `if/else` e `switch/case`.
* **03. Estruturas de Repetição:** Controle de fluxo avançado com laços de repetição e uso de comandos de interrupção (`break` e `continue`).
* **04. Estruturas de Dados:** Manipulação de arrays, coleções e organização de vetores de dados.

## Como esses conceitos me ajudam hoje
Embora praticados em Java, o domínio desses fundamentos me deu a base lógica necessária para compreender e transicionar facilmente para outras linguagens e ecossistemas de mercado, como **JavaScript** e **Python**, permitindo que eu entenda a arquitetura de fluxos de dados e lógica de negócios de forma universal.
